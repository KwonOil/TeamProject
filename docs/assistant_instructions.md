# 맞춤형 지침 요약

다음 항목은 사용자가 제공한 맞춤형 지침을 정리한 것입니다. 프로젝트 참여자가 저와 협업할 때 참고하면 됩니다.

## 1. 응답 스타일
- 모든 대답은 반드시 한국어 존댓말로 작성합니다.
- 실제 코드 구현부는 영어로 작성하지만, 주석과 설명은 한글로 작성해 깨지지 않도록 합니다.
- 코드에는 초보자도 이해할 수 있도록 상세한 주석을 포함합니다.

## 2. 개발 환경 가정
- 주요 개발 도구는 VSCode이며 사용 언어는 Python입니다.

## 3. 프로젝트 팀 구성
- 프로젝트는 로봇 팀, AI 팀, 서버 및 백엔드 팀, 대시보드 팀으로 구성되어 있습니다.
- 각 팀의 역할과 책임은 별도의 팀 가이드 문서(로봇, AI, 서버·백엔드, 대시보드 가이드)에 정리되어 있습니다.

## 4. 프로젝트 호칭
- "운전연수 프로젝트"라고 언급하는 경우 위 네 개의 팀 가이드라인을 기반으로 한 동일한 프로젝트를 의미합니다.
